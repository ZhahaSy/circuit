# 背景
对汽车售后手册的电路图开发流程进行改造，现在需要人工对照设计院图纸，设计。期望基于电路回路表实现可以使用代码生成电路图

## 电路图生成逻辑 （MVP版本）
目标：生成符合要求的电路图

渲染逻辑
普通元气件 在展示时，只需区分水平/垂直渲染即可

特殊元气件 电器盒 和 多入口电源元气件需要特殊定制，位置相对比较固定。




### 电路图绘制方向要求
电路图分为水平绘制/垂直绘制，根据不同的回路表区分不同的绘制方式

垂直绘制条件
1. 存在 电源入口，
2. 存在 主控制器。
3. 主控制器针脚相对平均

水平绘制条件
1. 电源电路图（只有单个主元件，其他的元件都是 省略式表达）


### 电路图布局点位规则
1. 多入口电源 在最上面/最左侧
2. 保险丝/继电器 包裹在 电器盒内部 
3. 控制器放在中间，如果当前电路图展示的非全部针脚，使用带有非全部针脚的控制器组件展示，控制器名称在控制器内部居中展示。可以在控制器相对位置的上，下两面分布插头
4. 线束&针脚：目标元气件， 除了（非线束类/省略式元气件/连接点 ）所有的线束都需要关联针脚。线束颜色规则：线束区分单色与复色，单色直接渲染单色，复色需要判断主色与次色，次色在主色中间。主线宽度为4px 复线宽度 为2px
5. 针脚展示规则：针脚根据线束走向，判定方向为 上，下，左，右。在针脚标签的右侧展示 针脚信息（例如BF01-1）
6. 对接插头：区别与单向插头，用于对接多个线束，展示时，根据线束走向，判定方向为 上，下，左，右，区分公母。分别对应不同的针脚
7. 接地：接地元气件与其针脚永远放在控制器下面
8. 元气件排版位置： 元气件位于所属控制器的上方或下方。相同或分类类似的元气件，临近排版。入口或出口相同的元气件 临近排版
